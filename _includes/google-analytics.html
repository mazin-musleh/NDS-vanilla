Consent: defaults to denied (Consent Mode v2). `_js/nds-cookies.js` reads `window.GA_TRACKING_ID` and calls `gtag('consent', 'update', …)` when the user accepts/declines the cookie banner. If a prior 'accepted' consent cookie exists on this load, we lift consent immediately so returning visitors are tracked. Perf: placed at end of body (not head), so the head parses faster and the inline cost is paid after most of the document is parsed. dataLayer + gtag + consent default are set synchronously at parse time so any deferred-bundle code (e.g. nds-cookies.js banner clicks) can call gtag() safely — calls queue to dataLayer and replay when gtag.js arrives. The gtag.js network fetch is injected via requestIdleCallback so it never competes during load. {%- endcomment -%} {%- if jekyll.environment == "production" and site.google_analytics and site.google_analytics != "" -%} {%- assign ga_id = site.google_analytics -%} {%- endif -%}
