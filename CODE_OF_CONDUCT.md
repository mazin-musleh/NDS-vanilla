# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct.

All contributors, issue reporters, and discussion participants are expected to follow it. The full text is maintained at the link above.
